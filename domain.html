Domain

Pengertian Domain

Domain adalah nama unik yang digunakan untuk mengidentifikasi server komputer dan memudahkan pengunjung mengakses website atau aplikasi online. Ini merupakan cara yang lebih mudah diingat dibandingkan menggunakan alamat IP.

Domain terdiri dari nama dan ekstensi. Contohnya, pada jagoanhosting.com, 'jagoanhosting' adalah nama domain dan '.com' adalah ekstensi domain.

Ekstensi domain terbagi menjadi tiga kategori: Top Level Domain (TLD), Second Level Domain (SLD), dan Third Level Domain (3LD).

Cara Kerja Domain

Ketika kamu memasukkan nama domain ke dalam browser, browser akan mengirimkan permintaan ke jaringan server global yang membentuk Domain Name System (DNS). Server di DNS, yang juga disebut nameserver, kemudian mencari lokasi website yang dimaksud.

Setelah lokasi website ditemukan, server akan mengirimkan data yang diperlukan untuk ditampilkan kepada pengguna.
